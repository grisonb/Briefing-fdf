- FDS / GAAR : mode d’écriture Pencil sur les PDF affichés.
- FDS / GAAR : annotations visuelles BFG au-dessus du PDF, sans modification du PDF source.
- FDS / GAAR : boutons Écrire PDF, Annuler trait, Effacer page.
- FDS / GAAR : ajout de la gomme.
- FDS / GAAR : choix de couleur rouge, bleu, vert, noir.
- FDS / GAAR : correction des carrés de couleur pour avoir une taille homogène.
- FDS / GAAR : ajout du curseur “Trait” pour régler l’épaisseur de 1 à 10 par pas de 0,5.

Changements déjà conservés depuis les versions précédentes :
- Connexion Accès Briefing valable jusqu’à 23h00 du jour de connexion.
- Authentification NAS en JSON HTTP 200 pour éviter les pages d’erreur Synology.

- Affichage compact du code de partage : “Code de partage généré : BFG-xxxx-xx”.
- Message de sauvegarde locale conservé avec le code de partage lorsqu’ils existent tous les deux.
- Clavier iPad non ouvert automatiquement au chargement de BFG.
- NOTAMs cochés / surlignés persistants si le même NOTAM reste présent et identique.
- Apple Pencil : surligneur NOTAMs spécifique BFG, avec scroll différencié de la sélection.
- GAAR : actualisation via relais NAS request-gaar-import.php.
- GAAR : zoom PDF légèrement réduit par rapport aux versions antérieures.
