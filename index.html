Aide : sauvegarde et partage du briefing

  1. Sauvegarder : enregistre localement le briefing affiché sur cet iPad/navigateur, avec les derniers PDF, NOTAMs, pélicandromes, cartes et données disponibles.
  2. Chargement automatique : à l’ouverture du briefing, la dernière sauvegarde locale de cet iPad/navigateur est chargée automatiquement si elle existe.
  3. Partager : crée un briefing partagé sur le NAS et fournit un code à ouvrir depuis un autre appareil.
  4. Effacer : réinitialise le stockage local de cet iPad pour BFG : sauvegarde locale, anciens PDF/NOTAM/TEMSI/SUP AIP, modes FDS/GAAR et caches PWA BFG. Les fichiers NAS et les briefings partagés déjà créés ne sont pas supprimés.
  5. Ouvrir un briefing partagé : permet de saisir un code du type BFG-JJMM-01 pour récupérer un briefing déjà partagé.

Tutoriel : Récupérer les NOTAMs sur NATS

  1. Connectez-vous sur le site NATS avec les identifiants suivants :
    • Login : Dash8
    • Mot de Passe : Civile-Feu26!
  2. Cliquez sur Pre-Flight Briefing.
  3. Puis sur Aerodrome PIB.
  4. Sous "Briefing Options", trouvez les 4 petits boutons. Cliquez sur celui avec une étoile (favoris), qui se trouve à droite du bouton +.
  5. Dans la liste qui apparaît, cliquez sur PELICANDROMES.
  6. Tout en bas de la page, cliquez sur le bouton Generate.
  7. Enfin, cliquez sur Print PDF pour télécharger le fichier.
  8. Enregistrez le PDF sur votre appareil, puis chargez-le sur cette page à l'aide du bouton "Charger PDF des NOTAMs".
